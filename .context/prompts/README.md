# AI Prompts

Pre-built prompts for common development tasks. Copy, customize, and use with your AI tool.

## How to Use

1. Choose the relevant prompt below
2. Replace `[placeholders]` with your specifics
3. Include relevant `.context/` files as context
4. Run with your AI tool
